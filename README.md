# Passwortgenerator

Ein einfaches Python-Projekt zur Generierung zufälliger Passwörter.

---

## Projektbeschreibung

Dieses Programm erstellt zufällige Passwörter mit einer frei wählbaren Länge. Dabei werden Groß- und Kleinbuchstaben, Zahlen sowie Sonderzeichen verwendet. Die Mindestlänge beträgt 16 Zeichen.

---

## Funktionen

- Generierung zufälliger Passwörter
- Verwendung von Groß- und Kleinbuchstaben, Zahlen und Sonderzeichen
- Frei wählbare Passwortlänge (ab 16 Zeichen)
- Überprüfung der Benutzereingabe
- Fehlerbehandlung bei falschen Eingaben
- Saubere und übersichtliche Ausgabe
